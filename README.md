🌐 Personal Portfolio Website

🚀 About the Project

This is a fully responsive and modern Personal Portfolio Website built using HTML, CSS, and JavaScript. It is designed to showcase my skills, projects, and journey as a developer in a clean and professional way.

The main goal of this project is to create a strong online presence where recruiters, collaborators, and peers can explore my work, technical skills, and achievements in one place.

✨ Features
🎯 Clean and modern UI/UX design
📱 Fully responsive (Mobile + Tablet + Desktop)
⚡ Fast loading and optimized performance
🎨 Smooth animations and transitions
📂 Project showcase with live/demo links
📬 Contact section for easy communication
🌙 (Optional) Dark mode support
🛠️ Tech Stack
HTML5 – Structure of the website
CSS3 – Styling and layout (Flexbox & Grid)
JavaScript (ES6) – Interactivity and dynamic behavior
